Commands

  • oi - Status
  • join - Sign up
  • leave - Cancel sign up
  • spec - Spectate

Instructions

New tournament is started each hour beginning with sign up's for ten minutes. When sign up close Tattoo will assign players to a team and launch the game. After the game Tattoo needs to know the score. When asking for it just type in the format redscore-bluescore, e.g. 6-2.

Now is time for next round. Tattoo will assign players to team by order of tournament result. #1 goes to red, #2 to blue, #3 red etc. Since this is repeated there will be one winner at the end.

Results

Current result is announced after each game and can also be viewed below.

Tournament can be found at Orbit

Tattoo can't handle duplicate names so make sure you join with a unique name.
