Instructions

Join the group and type 'oi'. To sign up type 'join', to cancel type 'leave', to spectate type 'spec'.

New tournament is started each hour. Five minutes before a group is created where you can sign up. When sign up close Tattoo will assign players to a team and launch the game. After the game Tattoo needs to know the score. When asking for it just type in the format redscore-bluescore, e.g. 6-2. Now is time for next round. Tattoo will assign players to team by order of tournament result. #1 goes to red, #2 to blue, #3 red etc. Since this is repeated there will be one winner at the end.

Current result is announced after each game and can also be viewed here. No problem reading raw json? Right? Good!

Tournament can be found at Orbit
