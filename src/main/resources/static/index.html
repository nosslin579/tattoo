Contact:Reddit or Github

Instructions

Tattoo is a fully automated TagPro tournament organizer. It will handle sign up, assign players to correct team and keep track om score. New tournament is started via configurable schedule. To know the score it uses http://tagpro.eu so make sure you have it installed. It can handle 8-12 sign ups, organized in a single group. If more then 8 players signed up there will be 3 - 5 qualification games. Points collected during these games will be carried over to the end games.

To join a tournament, join the group and type `join`. It is not possible to join as Some Ball so make sure you are logged in and have a unique name. If you get disconnected just join the group again. Current result is announced after each game and can also be viewed below.

Tournament can be found at https://tagpro.koalabeast.com/groups

Commands

  • join - Sign up
  • leave - Cancel sign up
  • spec - Spectate
  • poke - Status
